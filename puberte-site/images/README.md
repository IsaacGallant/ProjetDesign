Placez ici vos images (web-friendly: .jpg, .png, .webp). 

Usage :
- Déposez les fichiers dans le dossier `images/`.
- Dans les pages HTML, insérez `<img src="images/nom-de-fichier.jpg" alt="...">` ou ajoutez-les dans la section galerie de `index.html`.

Conseil : optimisez les images pour le web (taille et résolution) avant de les déposer.
